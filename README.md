# 🌿 Unani Herbal Healthcare Platform – Al Ghani

A professional **Unani & Herbal medicine website** built for a real client using **React + Vite**. Inspired by industry leaders like Hamdard & Qarshi — fully responsive, bilingual, and visually powerful.

## ✨ Features
- 🌐 **Bilingual Toggle** — Switch between English & Urdu instantly
- 🌿 **Animated Herb Orbit** — Rotating herbal ingredients around center logo
- 📱 **Fully Responsive** — Mobile, tablet & desktop ready
- 💊 **Diseases Section** — 12 treatments with real images
- 🧴 **Products Section** — Filterable herbal medicine catalog
- 📬 **Consultation Form** — Patient inquiry with disease selection
- 🔗 **WhatsApp & Social** — Floating contact buttons
- 🌙 **Component-Based** — Clean, modular React architecture

## 🛠️ Tech Stack
- **React + Vite**
- **CSS3** — Custom animations, transitions
- **Google Fonts** — Noto Nastaliq Urdu + Cinzel
- **Netlify** — Deployment

  
## 🖥️ Run Locally
npm install
npm run dev
```
